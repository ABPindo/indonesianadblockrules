ABPindo adalah daftar penapis iklan untuk situs berbahasa **Indonesia** dan **Malaysia**, yang dirancang untuk melengkapi penapis internasional seperti [EasyList](https://subscribe.adblockplus.org/?location=https://easylist.to/easylist/easylist.txt&title=Easylist) dan [AdGuard Base Filter](https://raw.githubusercontent.com/AdguardTeam/FiltersRegistry/master/filters/filter_2_Base/filter.txt).

ABPindo menggunakan sintaks [Adblock Plus](https://help.eyeo.com/en/adblockplus/how-to-write-filters) dan kompatibel dengan ekstensi seperti [uBlock Origin](https://github.com/gorhill/uBlock#installation), [uBlock Origin Lite](https://chromewebstore.google.com/detail/ublock-origin-lite/ddkjiahejlhfcafbddmgiahcphecmpfh), [AdGuard](https://adguard.com/en/adguard-browser-extension/overview.html), dan [Adblock Plus](https://adblockplus.org/). Kini ABPindo juga menyediakan penapis berbasis **hosts** dan **domain**, kompatibel dengan [AdAway](https://github.com/AdAway/AdAway), [Pi-Hole](https://github.com/pi-hole/pi-hole), [AdGuard Home](https://github.com/AdguardTeam/AdGuardHome), dan pemblokir DNS lainnya.

## Keuntungan

- **Lebih Bersih** — Tanpa iklan promosi berlebihan, iklan judi, dan iklan dewasa
- **Terhindar dari Penipuan** — Tanpa iklan yang menyamar sebagai tombol update, stream, atau download
- **Lebih Cepat** — Waktu membuka halaman web lebih singkat tanpa beban iklan

## Cara Menggunakan

### Untuk Ekstensi Browser

1. Pasang salah satu ekstensi berikut di peramban Anda:
   [uBlock Origin](https://github.com/gorhill/uBlock#installation) · [uBlock Origin Lite](https://chromewebstore.google.com/detail/ublock-origin-lite/ddkjiahejlhfcafbddmgiahcphecmpfh) · [AdGuard](https://adguard.com/en/adguard-browser-extension/overview.html) · [Adblock Plus](https://adblockplus.org/en/)

2. Klik tautan **Subscribe** untuk langsung mengaktifkan filter:
   **[▶ Subscribe ABPindo](https://subscribe.adblockplus.org/?location=https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/abpindo.txt&title=ABPindo)**

3. Atau, salin tautan **Raw File** dari tabel di bawah dan tambahkan secara manual melalui pengaturan filter di ekstensi Anda.

> **Catatan untuk pengguna uBlock Origin Lite:** Setel *default filtering mode* ke **Complete** agar semua elemen iklan terblokir. Lihat [panduan lengkapnya di sini](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/refs/heads/master/wiki/455486093-4bc27a5b-6dcb-406d-b46e-13092ece89c8.png).

> **Masalah jendela pop-up?** Baca [panduan memblokir popup](https://github.com/ABPindo/indonesianadblockrules/wiki/Panduan-Memblokir-Popup) di wiki.

### Untuk Pemblokir DNS (Sintaks Hosts & Domain)

| Sintaks | ABPindo | ABPindo + Adult | Contoh |
|---|---|---|---|
| Hosts file ([AdAway](https://github.com/AdAway/AdAway), [DNS66](https://f-droid.org/id/packages/org.jak_linux.dns66/), [NetGuard](https://netguard.me/), [personalDNSfilter](https://play.google.com/store/apps/details?id=dnsfilter.android&hl=id)) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/hosts.txt) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/hosts_adult.txt) | `0.0.0.0 example.com` |
| Domain ([Pi-Hole](https://github.com/pi-hole/pi-hole), [DNSCrypt](https://dnscrypt.info/)) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/domain.txt) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/domain_adult.txt) | `example.com` |
| Adblocker-syntax ([AdGuard Home](https://github.com/AdguardTeam/AdGuardHome), [Pi-Hole](https://github.com/pi-hole/pi-hole)) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/aghome.txt) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/aghome_adult.txt) | `\|\|example.com^` |
| [Dnsmasq (address)](https://thekelleys.org.uk/gitweb/?p=dnsmasq.git) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/dnsmasq.txt) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/dnsmasq_adult.txt) | `address=/example.com/0.0.0.0` |
| [Dnsmasq (server)](https://thekelleys.org.uk/gitweb/?p=dnsmasq.git) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/dnsmasq_server.txt) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/dnsmasq_adult_server.txt) | `server=/example.com/` |
| RPZ / [BIND](https://gitlab.isc.org/isc-projects/bind9.git) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/rpz.txt) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/rpz_adult.txt) | `example.com CNAME .` |
| [UNBOUND](https://github.com/NLnetLabs/unbound) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/unbound.txt) | [Raw](https://raw.githubusercontent.com/ABPindo/indonesianadblockrules/master/subscriptions/unbound_adult.txt) | `local-zone: "example.com" always_nxdomain` |

> **Catatan:** Penapis berbasis DNS bekerja secara berbeda dari ekstensi browser — pemblokiran dilakukan di level domain, bukan elemen halaman. Untuk menghindari false positive, cakupan default ABPindo versi DNS hanya mencakup `src/advert/adservers.txt` dan `src/advert/thirdparty.txt`. Versi `_adult` menambahkan `src/adult/adult_thirdparty.txt`.

## Berkontribusi 🤝

ABPindo adalah proyek komunitas. Jika menemukan situs yang terlewat, iklan baru yang tidak terblokir, atau blokir yang salah sasaran (*false positive*), Anda bisa membantu dengan cara berikut:

- **Laporkan masalah** melalui [GitHub Issues](https://github.com/ABPindo/indonesianadblockrules/issues) — sertakan URL halaman dan tangkapan layar jika memungkinkan
- **Kirim perbaikan langsung** melalui [Pull Request](https://github.com/ABPindo/indonesianadblockrules/pulls)

Sebelum membuat issue baru, pastikan masalah yang sama belum dilaporkan sebelumnya.
